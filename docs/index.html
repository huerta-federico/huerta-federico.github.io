Base de datos relacional en SQLite

Base de datos en SQLite diseñada e implementada para gestionar las operaciones centrales de un negocio de filtración de agua, incluyendo gestión de clientes, inventario, pedidos y mantenimientos. El proyecto incluye esquemas normalizados, integridad de datos mediante restricciones, triggers automatizados y vistas optimizadas para consultas, demostrando sólidos conocimientos en diseño de bases de datos y habilidades prácticas para la resolución de problemas.

Aplicación web con Flask, SQLite y Bootstrap

Aplicación web full-stack con Flask, Bootstrap & SQLite para gestionar los mantenimientos de los sistemas de filtración de agua de osmosis inversa, incluyendo el registro de clientes, direcciones, mantenimientos e inventario. Este proyecto incluye una BD normalizada, validación extensiva de datos, gestión de errores, diseño responsivo, arquitectura de múltiples capas, y programación orientada a objetos, y resalta la aplicación de estándares de diseño, desarrollo y documentación.

App móvil con Flutter y SQLite para Android

Aplicación móvil con Flutter para Android que me ayuda a manejar el programa de ejercicio físico 'Insanity Workout'. Me indica la rutina específica de cada día y marcarla como completada o saltada. Muestra el calendario del programa y registra estadísticas como el número de ejercicios completados, ciclos completados, etc. Incluye un formulario para las pruebas de rendimiento, y genera una gráfica con base en los resultados. Funciona mediante una base de datos local. Es responsiva y animada. Tiene funcionalidades dinámicas que se actualizan automaticamente. Permite exportar la base de datos como respaldo, e importarla posteriormente para restaurar los datos. La aplicación es un prototipo y estoy planeando añadir más funcionalidades y pulir el comportamiendo.
