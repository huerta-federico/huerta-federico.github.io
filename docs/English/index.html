Mobile app with Flutter and SQLite for Android

Mobile application with Flutter for Android that helps me manage the 'Insanity Workout' exercise program. It shows me the specific routine for each day and allows me to mark it as completed or skipped. It displays the program calendar and tracks statistics like the number of completed exercises, completed cycles, etc. It includes a form for fitness tests, and generates a chart based on the results. It works through a local database. It's responsive and animated. It has dynamic functionalities that update automatically. It allows exporting the database as a backup, and importing it later to restore the data. The application is a prototype and I'm planning to add more functionalities and polish the behavior.
